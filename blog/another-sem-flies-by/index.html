Another Sem flies by

Jul 16, 2021

Hi to everyone, again. I just completed my fourth semester. Yaay! I must say it kinda went well since I finally had core Computer Science courses. More importantly, I enjoyed the semester. I got to understand the computer a little bit more, but not enough. I guess I'll have to wait till my OS course in the coming semester.

Now, coming to the cool things that I did in the last few months:

  1. I've implemented text file compression using Huffman Encoding in C (oh boy, I spent way too much time on this one). 10/10 worth the effort.
  2. I converted a single-threaded matrix multiplication code into a multi-threadded one. There was a very significant increase in performance.
  3. I set up a home lab which consists of a Raspberry Pi 4B 2GB RAM (aka homepi) and an HP Z220 Workstation as a 2TB NAS. More about it in a separate blog (It's very pog yo!).
  4. I compiled the Xtended custom ROM for my phone, violet. Would try to be the official maintainer soon.


Another task that deserves a special mention: I started reverse-engineering the Windows driver of my laptop's fingerprint scanner, since there isn't an Open Source Linux driver for it. I plan on making a fprint compatible driver, once I understand how the device works. I also learnt a lot about how USB works along the way! I tried to write a basic driver using libusb, and note that its work in progress at the time of writing this. Here you go!

I also bought a mechanical keyboard, the HyperX Alloy Origins Core with blue switches. Typing feels great after that.

I shall try and keep this site as updated as possible hereafter, it's about time it happened
